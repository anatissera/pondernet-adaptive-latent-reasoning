## Summary
The gradient-checkpointing fix run: PonderNet with --gradient_checkpointing False so the latent loop's KV cache works during training. The fix established a clean training objective (the latent loop's KV cache works), validated by the surviving epochs landing at ~41% on the held-out validation split - on par with the 40.80% greedy validation baseline. The originally-headline 42.23% @ ep2 was on the **test** set (biased) and its checkpoint has since been **deleted, so it cannot be re-validated**.
